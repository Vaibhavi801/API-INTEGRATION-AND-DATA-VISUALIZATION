# API-INTEGRATION-AND-DATA-VISUALIZATION

#Weather Data Visualization Dashboard

#Project Overview
This project demonstrates how to fetch real-time weather data from a **public API (OpenWeatherMap)** using Python, process and clean the data, and create meaningful visualizations using **Matplotlib** and **Seaborn**.

The final output is a visualization dashboard that compares key weather parameters across multiple cities.


#Objectives
- Fetch live weather data from a public API
- Process and clean raw JSON data
- Convert data into a structured format using Pandas
- Create visualizations for weather analysis
- Build a single dashboard containing multiple plots


#Technologies Used
- **Python**
- **OpenWeatherMap API**
- **Requests** – API calls
- **Pandas** – Data processing
- **Matplotlib** – Plotting
- **Seaborn** – Styled visualizations
